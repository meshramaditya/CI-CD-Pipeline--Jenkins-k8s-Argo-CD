# Blue Cube — Cafe Admin Frontend

This is a minimal Next.js frontend for the Blue Cube cafe administrative dashboard. It includes:

- Login page (simple id/password)
- Dashboard with tabs: Orders, Billing, Transactions, Analytics
- Local persistence via `localStorage` for transactions

Run locally:

```powershell
cd Frontend
npm install
npm run dev
```

Open http://localhost:3000

Default credentials: `admin` / `bluecube`
